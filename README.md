 # Cost of Living Analysis in European Cities

## Синопсис
Этот проект посвящен анализу стоимости жизни в городах Европы на основе открытых данных.
Цель — выявить различия между городами и определить самые дорогие и доступные.

## Актуальность
Стоимость жизни — ключевой фактор при выборе города для жизни, учебы или работы.
Анализ помогает понять экономические различия между странами Европы.

## Исследовательские вопросы

- Какие города самые дорогие?
- Какие самые дешевые?
- Как отличается аренда жилья?
- Какие страны дороже в среднем?

## Данные
Источник данных: Numbeo (https://www.numbeo.com/cost-of-living/)

Были выполнены:
- удаление дубликатов
- удаление пустых значений
- проверка типов данных
- нормализация названий стран

## Анализ

Основные выводы:
- Самые дорогие города находятся в Западной Европе
- Восточная Европа значительно дешевле
- Аренда составляет значительную часть расходов
- Разница между странами существенная
